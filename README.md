# Look Up API

## Pre-requisite

- node 12.16.3
- npm 6.14
    
## Getting started

- git clone
- npm install
- npm run dev : Start local development server using nodemon


## Execution and writing of tests

- npm test : Start test server using mocha
- Using Chai and Chai-http with method *should*
